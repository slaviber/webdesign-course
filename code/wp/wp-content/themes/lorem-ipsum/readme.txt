INSTALLATION
============
1) Upload the "lorem-ipsum" folder this file is in to the wp-content/themes folder.
2) Login to your wordpress admin area and click on the "design" tab. You should see "Lorem Ipsum" under the list of available themes.
3) Click on the "Lorem Ipsum" image to install it as the current theme.
4) Optional - to be able to edit the theme from within the wordpress admin area, don't forget to change the file permissions (CHMOD) for the uploaded files to 777.
